Какие формы ведения бизнеса актуальны в Узбекистане?

Общество с ограниченной ответственностью (ООО)

Идеально подходит для малого и среднего бизнеса. ООО предоставляет учредителям ограниченную ответственность по обязательствам компании, что снижает риски.

Индивидуальный предприниматель (ИП)

Выбор для тех, кто хочет работать на себя без сложной структуры. Подходит для малого бизнеса и самозанятых граждан.

Акционерное общество (АО)

Предназначено для крупных проектов, связанных с привлечением инвестиций. Акции компании могут быть проданы, что упрощает привлечение капитала.

Частное предприятие (ЧП)

Удобный формат для небольших компаний с одним учредителем, который полностью контролирует бизнес.

Как выбрать подходящую форму?

  • ИП: для фрилансеров, частных мастеров, малого бизнеса.
  • ООО: для малых и средних компаний, заинтересованных в росте.
  • АО: для крупных бизнес-проектов и привлечения инвесторов.
  • ЧП: для индивидуальных владельцев бизнеса.

Какие документы нужны для регистрации бизнеса?

1

Для регистрации ООО, если участник — физическое лицо:

  • Скан-копии паспорта учредителей.
  • Электронное фото на белом фоне (3х4).
  • Нотариально заверенная доверенность на нашего сотрудника для проведения всех процедур.
2

Для регистрации ООО, если участник — юридическое лицо:

  • Выписка из торгового реестра.
  • Нотариально заверенная доверенность на представителя в Узбекистане.
  • Копии свидетельства о регистрации, устава компании и учредительного договора.
3

Дополнительная информация о новой компании:

  • Название компании.
  • Вид деятельности.
  • Налоговый режим.
  • Электронная почта, телефон, почтовый адрес с кадастровым номером.
  • Сумма уставного фонда.
  • Скан паспорта директора.
